Linux工具

发表于 2019-08-03 | 分类于 日常 ， Linux |
字数统计: 427 字 | 阅读时长 ≈ 1 分钟

Linux好用的一些工具，恩，我自己比较喜欢的,能够极大的提高我的工作效率的

命令行工具

tmux

最好用的终端复用器

ag

比grep，ack更快的递归搜索文件内容

tig

字符模式下交互查看git项目，可以用来代替git命令

mycli

mysql的客户端，支持语法高亮，命令补全，Python开发

jq

json文件的格式化工具

shellcheck

shell脚本的静态检查工具

mosh

基于UDP的终端连接，可以代替ssh

fzf

命令行下的模糊搜索工具

htop/glances

更美观的进程监控工具，用于代替top命令，glances好像要更强大一些

axel

多线程的下载工具

sz/rz

交互式文件传输，非常适合在多重跳板机下传输文件，不需要一级一级的传输

cloc

代码统计工具

script/scriptreplay

终端会话录制工具

lcdiff

分屏显示diff，用于替换diff

ranger/mc

内容预览/浏览工具

multitail

更好用的tail -f 工具

thefuck

命令行更正工具

tldr

too long didn’t read

figlet

demo制作工具

mackup

应用偏好设置备份和同步工具

cv

显示进度条的cp命令，用于替换cp

wtf

娱乐小工具，用于显示缩略简写

reptyr

重新运行终端的任务

pm2

用于在后台运行和管理进程

httpie

用于代替的curl的http客户端工具

transfer.sh

可以通过简单的命令行来实现文件共享操作

stegify

可以将任意文件隐藏在图片中

spacevim

莫名的即使感，以前我用的是spacemacs

aptitude

用于代替apt的工具

redshift

护眼工具

fx

可以展开的json格式化工具
